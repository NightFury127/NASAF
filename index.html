Getting Started

Learn how to upload data and get your first exoplanet analysis results

Quick Start Guide

  1. Upload Your Data: Drag and drop CSV files containing light curve data or use our manual parameter input
  2. AI Analysis: Our ensemble AI model analyzes your data using Random Forest, CNN, and advanced algorithms
  3. Review Results: Get detailed classification results with confidence scores and scientific explanations
  4. Explore & Compare: Use our database to find similar exoplanets and explore the 3D visualization

Data Formats

Supported file formats and data structure requirements

Supported Formats

  • CSV Files: Comma-separated values with time series data
  • FITS Files: Flexible Image Transport System files

Required Columns

  • Time (BJD or MJD format)
  • Flux or Magnitude values
  • Error values (optional but recommended)

AI Models

Understanding our machine learning algorithms and accuracy metrics

Ensemble Classifier

Our primary model combines Random Forest, Convolutional Neural Networks, and Support Vector Machines for maximum accuracy (99.6%).

Model Features

  • Cross-validation with 5-fold methodology
  • Uncertainty quantification
  • Feature importance analysis
  • Explainable AI outputs

Results Interpretation

How to read and understand your analysis results

Classification Types

  • Confirmed: High confidence detection (>95% accuracy)
  • Candidate: Potential detection requiring follow-up (80-95% accuracy)
  • False Positive: Signal likely from stellar activity or noise

Confidence Scores

Our confidence scores represent the model's certainty in the classification, based on statistical analysis of signal characteristics.
